Overview

Before a dataset is ingested into the Pilot MAAP, the data provider is required to fill out this questionnaire. Information provided in this questionnaire will be used to create metadata for the dataset. This form consists of multiple pages containing questions organized by topic. Data providers are required to complete most questions in order for the MAAP publication process to occur.

Asterisks in the questionnaire indicate required questions. Please note that the more information provided, the more complete the metadata will be.

A unique identifier will be provided for each new record. Please save the number to refer back to the questionnaire you are filling.

You may contact the MAAP data team with any questions you may have about the form to Kaylin.
